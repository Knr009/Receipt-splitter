# Receipt-splitter

A minimal React/Vite application for splitting a receipt among people. This repo only contains the component logic; the environment needs to be set up before running.

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```
2. **Start the development server**
   ```bash
   npm run dev
   ```
3. Open <http://localhost:5173> in your browser to see the app.

The UI features the ability to add people, items, assign splits, and calculate tax/tip proportions.
